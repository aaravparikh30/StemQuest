# STEMQuest

This website showcases various research opportunities available at the University of Utah. It provides a platform for students and researchers to explore and connect with exciting research projects across different disciplines. Whether you are seeking hands-on research experience or looking for collaborators, this website aims to streamline the process of finding and applying for research opportunities.

## Features

- **About Section**: Provides an overview of the website and its purpose.
- **Why This Website Section**: Highlights the benefits of using this platform for finding and applying to research opportunities.
- **Available Opportunities Section**: Displays a list of available research opportunities, including their titles, descriptions, and contact information.
- **Contextual Information Section**: Provides additional information about how to stay informed, apply to opportunities, and stay updated with the latest research projects.

## Getting Started

1. Clone the repository or download the code.
2. Customize the code as per your requirements, such as updating the university logo, adding more research opportunities, or modifying the color palette.
3. Replace the `university_logo.png` file with your own logo, ensuring it is located in the same directory as the HTML file.
4. Open the HTML file (`index.html`) in a web browser to view the website locally.

## Customization

- **Logo**: Replace the `university_logo.png` file with your own logo, ensuring it is named the same and located in the same directory as the HTML file.
- **Color Palette**: Customize the color palette by modifying the HEX color codes in the CSS section of the HTML file.
- **Content**: Modify the text content in the HTML file to reflect your own research opportunities and relevant information.
- **Structure**: Update the HTML structure and layout as needed to match your design preferences.
